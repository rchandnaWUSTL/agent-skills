# Amazon Kiro Instructions for Terraform

This repo contains instructions files for the Amazon Kiro IDE when working on Terraform providers.

Currently supported:

- hooks
  - create-new-action-spec
- specs
  - aws_dynamodb_create_backup-action (as an example)
- steering
  - actions

## Usage

1. Copy files from this repo's `.kiro` directory into your project's root directory.
2. In Kiro, click on the Kiro tab, then on each hook, spec, or steering file to use it.
